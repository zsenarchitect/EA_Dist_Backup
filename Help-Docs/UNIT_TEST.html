UNIT_TEST

Test function for a module with a dictionary of test cases. Only intended to run in terminal, using Python 3. Rhino and Revit in-envionment testing will be supported with future updates. Args: test_dict (dict): Dictionary of test cases. filename (str): Filename of the module to test. Use the following formats for test_dict: test_dict = { "function_name_1": { "'string_arg_1'": expected_result, "'string_arg_2'": expected_result, ... }, "function_name_2": { "num1, num2": expected_result, "num3, num4": expected_result, ... }, "function_name_3": { "[list_arg_1]": expected_result, ... }, "function_name_4": { "{'string_arg', num_arg}": expected_result, ... ... } Returns: bool: True if all tests pass, False if any test fails.

EnneadTab

WEB

Web-related functionality for EnneadTab. This module provides web-related utilities including: - Documentation to HTML conversion - Image embedding - HTML template generation

Functions

documentation2html(doc_data_list, html_path)

Generates an HTML file with embedded images from documentation data. Args: doc_data_list (list): List of documentation data dictionaries html_path (str): Path to save the HTML file Each doc_data dictionary should contain: - alias: Command name - doc: Tooltip text - icon: Path to icon image (optional)

SECRET

Utilities for referencing secrets, such as API keys and developer identity information.

Functions

get_api_key(app_name)

Returns the API key for the specified app. Accepted keys: "EnneadTabAPI" # "chatgpt_api_key" # "translator_api_key" # "reporter_api_key" # "clone_helper" # "miro_oauth" Args: app_name (string): The name of the app to get the API key for. Returns: string: The API key for the specified app.

get_dev_info(developer_name, key)

Get developer information from the secret file. Args: developer_name (string): The name of the developer. key (string): The key to get the value for. Returns: string: The value of the key for the developer.

get_dev_dict()

Get the dictionary of developers from the secret file. Returns: dict: The dictionary of developers.

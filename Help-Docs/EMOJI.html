EMOJI

Get emojis from the emoji library.

Functions

get_all_emojis()

Get all emojis from the emoji library. Returns: list: List of emojis. Returns empty list if there's an error reading the emoji file.

pick_emoji_text()

Pick an emoji text from the displayed list and copy it to the clipboard.

random_emoji()

Pick a random emoji. Returns: str: A random emoji character, or empty string if no emojis are available.
